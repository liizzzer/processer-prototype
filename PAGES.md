## 1. Список протоколов
**Маршрут:** `/`
**Figma node:** `21:3293`
**Статус:** ✅ Готово

### Содержимое
- Сайдбар (активный пункт: Start)
- Заголовок: "Protocols" + счётчик "12 protocols"
- Кнопка "New protocol"
- Аватар пользователя + имя
- Поиск по протоколам
- Фильтр "All protocols" + кнопка "Filters"
- Сетка карточек протоколов (3 колонки)

### Карточка протокола
- Бейдж статуса (Pending approval — оранжевый)
- Иконки: редактировать, меню
- Название протокола (ALL CAPS)
- Версия + дата
- Краткое описание (3 строки, обрезается)
- Клик по карточке → переход на `/protocols/[id]`

## 2. Просмотр протокола
**Маршрут:** `/protocols/[id]`
**Figma node:** `154:1497`
**Статус:** ✅ Готово

### Содержимое
- Сайдбар (активный пункт: New protocol)
- Кнопка назад ←
- Заголовок протокола + версия/дата
- Кнопки "Edit" → `/protocols/[id]/edit` и "Print"
- **Табы:** Details · People · RACI · Diagram · Changes

### Таб: Details
- Блок PURPOSE + EXPECTED RESULT (двухколоночный, bg-muted, скруглённый)
- Карточка "Expected activities":
  - Кнопка "Open in diagram"
  - Список шагов (нумерованные тёмные кружки):
    - Название + описание
    - Бейдж "Subprocess" (пунктирная рамка) если применимо
    - Метаданные: владелец, инструменты, входящий/исходящий документ (иконки)

### Таб: People
- Карточка "Behaviours"
- Секции: Leaders / Managers / Associates
- Аватары с именем и ролью

### Таб: RACI
- Легенда: R (тёмный) / A (teal) / C (amber) / I (серый)
- Таблица: активности × участники, цветные бейджи

### Таб: Diagram
- ⬜ Заглушка — требует отдельной библиотеки диаграмм

### Таб: Changes
- ⬜ Заглушка

## 3. Редактирование протокола
**Маршрут:** `/protocols/[id]/edit`
**Figma node:** `154:2808`
**Статус:** ✅ Готово

### Содержимое
- Сайдбар (активный пункт: New protocol)
- Кнопка назад ←
- Заголовок "Edit | [название]" + версия/дата
- Карточка "Process details":
  - Поле: Название протокола
  - Поля: Ожидаемый результат + Цель (textarea, 2 колонки)
- Карточка "Activities":
  - Кнопка "+ Add step"
  - Шаги: активность, владелец, стейкхолдеры + роли, инструменты, документ, заметки
  - Кнопка удалить шаг
- Нижняя панель: Cancel · Save as Draft · Save and submit for approval
